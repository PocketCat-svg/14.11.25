 Принцип работы всего кода:
Открываем базу данных → создаем таблицу

Открываем JSON файл → читаем данные студентов

Для каждого студента из JSON → добавляем его в таблицу базы данных

Сохраняем изменения → закрываем соединение


Интерактивный режим Python
Открой терминал в VS Code

Введи команду:

bash
python
Теперь ты в интерактивном режиме Python (увидишь >>>)

Импортируй и запусти функции:

python
# Импортируем весь код
exec(open('database.py').read())

# Теперь можно вызывать функции:
read_students()
create_student("Тестовый Студент", 25, "Биология")
read_students()
